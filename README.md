# Visualizador de Árboles y Recursividad

Este proyecto consiste en un sistema interactivo que permite comprender el funcionamiento de los árboles binarios y la recursividad mediante la visualización de distintas operaciones sobre la estructura. El programa permite trabajar con árboles binarios, árboles binarios de búsqueda (BST) y árboles AVL, mostrando de forma gráfica cómo se insertan, buscan y eliminan nodos dentro del árbol. Además, incluye recorridos como preorden, inorden y postorden, permitiendo observar el orden en que los nodos son visitados. En el caso de los árboles AVL, el sistema también muestra el proceso de balanceo y las rotaciones necesarias para mantener el equilibrio de la estructura.  

El objetivo principal del proyecto es facilitar el aprendizaje de temas relacionados con estructuras de datos, especialmente árboles binarios, árboles BST, árboles AVL, recursividad y visualización de algoritmos, de una manera gráfica, interactiva y fácil de comprender.

## Funcionalidades específicas

### Inserción de nodos
Permite agregar valores al árbol.  
En los árboles BST y AVL, los nodos se colocan automáticamente en la posición correcta.

### Búsqueda de valores
Busca un nodo dentro del árbol mostrando el recorrido realizado hasta encontrarlo.

### Eliminación de nodos
El sistema maneja los distintos casos de eliminación:

- Nodo hoja
- Nodo con un hijo
- Nodo con dos hijos

### Recorridos
Se incluyen los recorridos:

- Preorden
- Inorden
- Postorden

Mostrando visualmente el orden en que los nodos son visitados.

### Árbol AVL
El árbol AVL se balancea automáticamente mediante:

- Rotaciones simples
- Rotaciones dobles
- Cálculo de altura
- Factor de balance

### Guardar y cargar árboles
Permite almacenar estructuras trabajadas para seguir utilizándolas más adelante.

### Información del árbol
El sistema muestra información general como:

- Altura del árbol
- Cantidad de nodos
- Valor de la raíz
